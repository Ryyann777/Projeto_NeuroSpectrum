### Sistema de Gerenciamento para Clínica de Psicologia - Espectro Autista

*NeuroSpectrum* é um sistema completo de gerenciamento para clínicas especializadas em neurodesenvolvimento, com foco em Transtorno do Espectro Autista (TEA).

---

## ✨ Funcionalidades

- Login seguro com autenticação social (Google/Apple)
- Dashboard com métricas da clínica
- Cadastro completo de pacientes
- Agendamento e visualização de consultas com calendário
- Emissão de laudos psicológicos
- Relatórios e recursos

---

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Design Responsivo
