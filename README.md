# 🧠 NeuroSpectrum

Sistema de Gerenciamento para Clínica de Psicologia especializada no Transtorno do Espectro Autista (TEA).

---

## 📋 Sobre o Projeto

O **NeuroSpectrum** é um sistema web completo de gerenciamento para clínicas especializadas em neurodesenvolvimento, com foco em pacientes com Transtorno do Espectro Autista (TEA). O projeto foi desenvolvido como trabalho acadêmico utilizando tecnologias front-end.

---

## ✨ Funcionalidades

- 🔐 Login seguro com autenticação social (Google/Apple)
- 📊 Dashboard com métricas da clínica
- 👤 Cadastro completo de pacientes
- 📅 Agendamento e visualização de consultas com calendário
- 📄 Emissão de laudos psicológicos
- 📈 Relatórios e recursos

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Design Responsivo

## 🚀 Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/Ryyann777/Projeto_NeuroSpectrum.git
```

2. Acesse a pasta do projeto:
```bash
cd Projeto_NeuroSpectrum/neurospectrum-site/neurospectrum
```

3. Abra o arquivo `index.html` no seu navegador.

> Não é necessário instalar nenhuma dependência — o projeto roda diretamente no navegador!
